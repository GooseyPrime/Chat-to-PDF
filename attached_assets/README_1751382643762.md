# AI Chat PDF Exporter (Replit Edition)

Export any public ChatGPT, Claude, or Gemini share link to a clean, readable PDF.

## 🧾 Features
- Clean UI for entering any AI chat share URL
- Uses Puppeteer with headless Chrome to capture clean PDF output
- Supports ChatGPT, Claude.ai, and Gemini links
- Hosted via Express.js backend

## 🛠 Setup on Replit
1. Create a new Node.js project
2. Add `express`, `puppeteer-core`, and `chrome-launcher` to `package.json` or install via shell:

```bash
npm install express puppeteer-core chrome-launcher
```

3. Place all files in your Replit project
4. Run `server.js`

## 📎 Usage
- Open the web interface
- Paste a public AI chat link (e.g. ChatGPT share link)
- Hit "Download PDF"

## ⚠️ Notes
- Gemini and Claude formatting may require further fine-tuning.
- Make sure the Replit project has access to Chrome through `chrome-launcher`.
